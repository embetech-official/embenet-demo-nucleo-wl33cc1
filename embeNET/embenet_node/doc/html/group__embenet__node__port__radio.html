Radio Interface

Detailed Description

Provides interface to the radio transceiver.

◆ EMBENET_RADIO_TxEnable()

EMBENET_RADIO_Status EMBENET_RADIO_TxEnable ( EMBENET_RADIO_Channel  channel,
EMBENET_RADIO_Power  txp,
uint8_t const *  psdu,
size_t  psduLen 
)

Wakes transceiver from IDLE state.

Prepares transceiver for transmission. Load new data (packet) to receiver's buffer/fifo.

Parameters
[in]channel- Channel number.
[in]txptransmit power in dBm
[in]psdu- data pointer, must not be NULL
[in]psduLen- data length in bytes (must be in range EMBENET_RADIO_MIN_PSDU_LENGTH to EMBENET_RADIO_MAX_PSDU_LENGTH)
Return values
EMBENET_RADIO_STATUS_SUCCESSon success
EMBENET_RADIO_STATUS_PARAMETER_ARGS_OUT_OF_BOUNDSwhen psduLen < EMBENET_RADIO_MIN_PSDU_LENGTH or psduLen > EMBENET_RADIO_MAX_PSDU_LENGTH
EMBENET_RADIO_STATUS_COMMUNICATION_ERRORon error
